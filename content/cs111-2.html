Syllabus template for Programming Fundamentals Laboratory taught at Rutgers University - Camden. The syllabus for CS 111-1 can be found on the course website.

Learning goal(s): Practice programming assignments based on weekly class material. Students will become familiar with applying fundamental programming concepts in Python. In addition to the topics covered during class, students will also learn to work with the command line, organize their work, version control (preferably with git and/or GitHub), run programs via the shell and safely debug and test their code.

Meeting days (online): M (LECTURE), W (LECTURE), F (LAB); 12:30-1:50 PM.

Before joining:

  1. - Follow the Python installation guide to set up Python version 3.7.3 or greater.
  2. - Fill out the pre-class survey. Note: to access this form, you must sign-in with your Rutgers accounts.
